# 🖥️ Windows 11 入門テキスト（PC初心者向け）

## 📘 1. Windows 11とは  
Windows 11（ウィンドウズ イレブン）は、Microsoft が提供する **OS（オペレーティングシステム）** です。  
OSはパソコンを動かすための「土台」で、アプリを動かしたり、ファイルを管理したりする役割を持っています。

---

# 📁 基本用語の説明

## ● フォルダ（Folder）
フォルダは **ファイルを整理するための入れ物** です。  
写真・文書・音楽などをまとめて管理できます。
ディレクトリと表現することもあります。

---

## ● ファイル（File）
ファイルは **実際のデータそのもの** です。  
文書、写真、動画、音楽など、すべてファイルとして扱われます。

---

## ● デスクトップ（Desktop）
パソコンを起動したときに最初に表示される画面です。  
よく使うファイルやアプリのショートカットを置くことができます。

---

## ● タスクバー（Taskbar）
画面下にある横長のバーです。  
アプリの起動、切り替え、時刻の確認などができます。

---

## ● スタートメニュー（Start Menu）
タスクバーの中央にある Windows マークをクリックすると開くメニューです。  
アプリ一覧、設定、電源操作（シャットダウンなど）ができます。

---

## ● エクスプローラー（Explorer）
フォルダやファイルを開いたり整理したりするためのアプリです。  
黄色いフォルダのアイコンが目印です。

---

# 🧩 アプリとウィンドウの概念

## ● アプリ（Application / App）
アプリとは **特定の目的のために動くソフトウェア** のことです。

例：  
- 文書を作る → Word  
- 表計算 → Excel  
- インターネットを見る → Edge / Chrome  
- 写真を見る → フォト  
- ファイルを管理する → エクスプローラー  

スマホのアプリと同じイメージでOKです。  
Windows ではアプリを **タスクバー** や **スタートメニュー** から起動します。

---

## ● ウィンドウ（Window）
ウィンドウとは、**アプリが表示される枠（画面）** のことです。

アプリを開くと、四角い枠が表示されますよね？  
あれが「ウィンドウ」です。

ウィンドウには次のような操作ができます：

- **移動**：上のバーをドラッグして好きな位置へ動かす  
- **サイズ変更**：端をドラッグして大きさを変える  
- **最小化**：タスクバーにしまう  
- **最大化**：画面いっぱいに広げる  
- **閉じる**：アプリを終了する  

### 🔍 アプリとウィンドウの関係  
- **アプリ＝中身（ソフト）**  
- **ウィンドウ＝そのアプリを表示する枠**  

スマホで例えるなら：  
- アプリ → 写真アプリそのもの  
- ウィンドウ → 写真アプリを開いたときの画面  

Windows では複数のウィンドウを同時に開けるため、  
「Word とブラウザを並べて作業する」  
といったことができます。

---

# 🧭 Windows 11 の基本操作

## 1. アプリを開く  
- タスクバーのアイコンをクリック  
- スタートメニューから探してクリック  

## 2. ファイルを開く  
- エクスプローラーを開く  
- フォルダをたどって目的のファイルをダブルクリック  

## 3. ウィンドウを切り替える  
- タスクバーのアイコンをクリック  
- Alt + Tab キーで切り替え  

---

# 💡 初心者がまず覚えると便利なポイント

- **アプリはタスクバーやスタートメニューから起動できる**

- **複数のウィンドウを並べて作業できるのがPCの強み**
