# 🖥️ Windows初心者向け操作テキスト  
## ショートカットキー & コピー＆ペースト入門

---

# 📘 1. ショートカットキーとは  
ショートカットキーとは、**キーボードの特定の組み合わせで操作を素早く行う方法**です。

例：  
- マウスで右クリック → コピー  
- ショートカットキー → **Ctrl + C**

同じ操作でも、ショートカットを使うと **作業が速く、正確** になります。

---

# ✂️ 2. コピー & ペーストの基本

## ● コピー（Ctrl + C）
選んだ文字やファイルを「一時的に保存」します。

## ● ペースト（Ctrl + V）
コピーした内容を貼り付けます。

## ● 切り取り（Ctrl + X）
コピーと似ていますが、**元の場所から移動**させたいときに使います。

---

## 🔧 コピー＆ペーストの練習例

### ① 文字をコピーする  
1. コピーしたい文字をドラッグして選択  
2. **Ctrl + C**  
3. 貼り付けたい場所をクリック  
4. **Ctrl + V**

### ② ファイルをコピーする  
1. ファイルをクリックして選択  
2. **Ctrl + C**  
3. 別のフォルダを開く  
4. **Ctrl + V**

### ③ ファイルを移動する（切り取り）  
1. ファイルを選択  
2. **Ctrl + X**  
3. 移動先のフォルダで **Ctrl + V**

---

# ⌨️ 3. よく使うショートカットキー一覧

## ■ 基本操作
- **Ctrl + C**：コピー  
- **Ctrl + V**：ペースト  
- **Ctrl + X**：切り取り  
- **Ctrl + Z**：元に戻す（Undo）  
- **Ctrl + A**：すべて選択  
- **Ctrl + S**：保存  

---

## ■ ウィンドウ操作
- **Alt + Tab**：開いているウィンドウを切り替える  
- **Windowsキー + D**：デスクトップを表示  
- **Windowsキー + ↑**：ウィンドウを最大化  
- **Windowsキー + ↓**：ウィンドウを最小化  

---

## ■ 文字入力に便利
- **Ctrl + Backspace**：単語ごと削除  
- **Shift + 方向キー**：文字を選択  
- **Ctrl + 方向キー**：単語単位で移動  

---

# 📁 4. ドラッグ＆ドロップとの違い

| 操作方法 | 特徴 |
|---------|------|
| **コピー＆ペースト** | 正確・確実。離れた場所への移動に向く |
| **ドラッグ＆ドロップ** | マウス操作だけで直感的。ただし誤操作が起きやすい |

初心者には **コピー＆ペーストの方が安全** です。

---

# 🧭 5. 初心者がつまずきやすいポイント

- コピーしたつもりで **選択が外れている**  
- ペースト先の場所を **クリックしていない**  
- 切り取り（Ctrl + X）とコピー（Ctrl + C）を混同  
- Alt + Tab でウィンドウが切り替わることを知らない  
- 「Ctrlキーを押しながら C」を同時に押す感覚に慣れていない
