# 🖼 PC初心者向け：スクリーンショットを撮って画像として保存する方法  
（Windows 11）

## 1. スクリーンショットとは  
今見えている画面を「写真のようにそのまま画像として残す」機能です。  
メモ代わりに残したり、操作を人に伝えるときに便利です。

---

# 2. 基本の撮影方法（全画面）

### ● 全画面を撮影する  
キーボードの **「PrtScn（Print Screen）」キー** を押します。

- 画面全体がコピーされます  
- まだ画像ファイルにはなっていません  
- クリップボード（コピーの一時保存場所）に入った状態です

---

# 3. ペイントを開く  
撮影した画像を保存するために「ペイント」を使います。

### ペイントの開き方  
1. **スタートボタン** をクリック  
2. 検索欄に **「ペイント」** と入力  
3. 表示された「ペイント」をクリック

---

# 4. ペイントに貼り付ける  
ペイントが開いたら、撮影した画像を貼り付けます。

- **Ctrl + V**  
または  
- 左上の「貼り付け」ボタン

貼り付けると、撮影した画面が表示されます。

---

# 5. 画像として保存する

1. 左上の **「ファイル」**  
2. **「名前を付けて保存」**  
3. 保存形式は **PNG** が推奨  
4. 保存場所を選ぶ（例：ピクチャ）  
5. ファイル名を入力  
6. **「保存」**

---

# 6. 範囲を選んで撮影する（部分だけ撮りたい場合）

### ● 範囲指定スクリーンショット  
**Windowsキー + Shift + S**

画面が少し暗くなり、上部にツールが表示されます。

選べるモード  
- **四角形の範囲**（ドラッグして選ぶ）  
- **フリーフォーム**（自由な形）  
- **ウィンドウ**（特定のウィンドウをクリック）  
- **全画面**

撮影後、右下に小さなプレビューが出ます。  
→ クリックすると編集画面（Snipping Tool）が開き、保存できます。  
→ そのまま **Ctrl + V** でペイントに貼り付けてもOK。

---

# 7. アクティブウィンドウだけ撮影する  
（今選んでいるウィンドウだけ撮りたい場合）

### ● アクティブウィンドウ撮影  
**Alt + PrtScn**

- 今前面にあるウィンドウだけがコピーされます  
- ペイントに **Ctrl + V** で貼り付けて保存できます

---

# 8. 自動で画像ファイルとして保存する（最速の方法）

### ● 全画面を自動保存  
**Windowsキー + PrtScn**

- 画面が一瞬暗くなります  
- 自動で PNG ファイルとして保存されます  
- 保存場所：  
  `ピクチャ > スクリーンショット`  
  例：  
  `C:\Users\<ユーザー名>\Pictures\Screenshots\`

ペイントを使わず、**キーを押すだけで画像ファイルができる**のでとても便利です。

---

# 📘 まとめ（どの方法を使うか）

| 撮影したい内容 | 操作 |
|----------------|------|
| 画面全体 | PrtScn |
| 範囲を選びたい | Win + Shift + S |
| ウィンドウだけ | Alt + PrtScn |
| 自動で保存したい | Win + PrtScn |
| ペイントで編集・保存 | Ctrl + V → 名前を付けて保存 |
