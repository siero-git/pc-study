# 📘 1. Excelとは  
Excel（エクセル）は、Microsoft が提供する **表計算ソフト**です。

できることの例：  
- 表を作る  
- 計算する（合計・平均など）  
- グラフを作る  
- データを整理する  

会社の事務作業で最もよく使われるアプリのひとつです。

---

# 🖥️ 2. Excelを起動する

## ● 方法①：タスクバーから開く  
画面下のタスクバーに Excel のアイコンがあればクリック。

## ● 方法②：スタートメニューから開く  
1. Windowsマークをクリック  
2. アプリ一覧から「Excel」を選ぶ  

---

# 📄 3. Excelの画面構成

Excelを開くと、次のような部分があります。

### ● セル（Cell）  
表の「マス目」1つ1つのこと。  
文字や数字を入力する場所。

### ● 行（Row）  
横方向の並び。1, 2, 3… と番号が付いている。

### ● 列（Column）  
縦方向の並び。A, B, C… とアルファベットが付いている。

### ● シート（Sheet）  
1つのファイルの中にある「ページ」のようなもの。

---

# ✏️ 4. 文字や数字を入力する

1. 入れたいセルをクリック  
2. キーボードで文字を入力  
3. Enterキーで確定  

※セルを移動しながら入力したい場合は「Enter」や「Tab」で移動できます。

---

# 📊 5. 表を作ってみよう（練習）

例：買い物リスト

| A列 | B列 |
|-----|-----|
| 品名 | 金額 |
| りんご | 120 |
| バナナ | 150 |

### 入力手順  
1. A1 に「品名」  
2. B1 に「金額」  
3. A2 に「りんご」  
4. B2 に「120」  
5. A3 に「バナナ」  
6. B3 に「150」

---

# ➕ 6. 計算してみよう（合計）

1. 合計を表示したいセル（例：B4）をクリック  
2. `=SUM(B2:B3)` と入力  
3. Enterキーで確定  

→ B2 と B3 の合計が表示されます。

※初心者には「オートSUM（Σ）」ボタンを使うと簡単です。

---

# 🎨 7. 表を見やすくする（書式設定）

### ● 太字にする  
選択 → **Ctrl + B**

### ● 文字の色を変える  
上のメニュー「ホーム」→「フォントの色」

### ● セルに枠線を付ける  
「ホーム」→「罫線」

### ● 列の幅を広げる  
列の境界をドラッグして調整

---

# 💾 8. ファイルを保存する

1. 左上の「保存」アイコンをクリック  
2. 保存場所（ドキュメントなど）を選ぶ  
3. ファイル名を入力  
4. 「保存」

※ショートカット：**Ctrl + S**

---

# 🔄 9. よく使うショートカット

| 操作 | ショートカット |
|------|----------------|
| コピー | Ctrl + C |
| 貼り付け | Ctrl + V |
| 切り取り | Ctrl + X |
| 元に戻す | Ctrl + Z |
| 保存 | Ctrl + S |
| すべて選択 | Ctrl + A |

---

# 🧭 10. 初心者がつまずきやすいポイント

- セルを選択していない状態で入力しようとする  
- 数字を入力しても計算されない（＝式を入れていない）  
- 列幅が狭くて「####」と表示される  
- 保存場所が分からなくなる  
- Enterで確定しないと入力が反映されない
